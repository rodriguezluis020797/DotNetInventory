# .NET Inventory Application

A simple .NET 10 MVC inventory tracking application. Every architectural decision is documented in the accompanying blog series.

## Tech Stack

- **Framework** — .NET 10 MVC
- **Database** — SQL Server
- **ORM** — Entity Framework Core 10
- **Auth** — ASP.NET Core Identity + JWT with refresh tokens
- **Language** — C# 14
